# DpyBot-MusicCog

Music cog for https://github.com/Jackenmen/DpyBot using Lavalink.py and Lavalink V4.
Based on https://github.com/devoxin/Lavalink.py/blob/0f6b0a2cead8ee781b16f25e4c9dc5f9401ee5dc/examples/music.py.

You will need to run `pip install -U "lavalink~=5.10.0"` before installing the cog from this repository.

This cog requires you to run Lavalink V4 in a separate process with the following configuration:

-   Host: `localhost`
-   Port: `2333`
-   Password: `youshallnotpass`

You can find an example config in the [`application.yml` file](./application.yml).
